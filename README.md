# Joris Röling

Indie audio software at [Bonboa](https://bonboa.com), Naarden NL.

I build instruments and tooling that sit between DAWs, MIDI hardware, and lately LLMs —
JUCE/CLAP/VST3 plugins, Bitwig controller scripts, the occasional CLI.

## Currently

**Encore** — a live generative composer. Claude composes a "Frame" of musical state; a local
sequencer plays it. JUCE host, u-he Hive as the MVP voice. Private for now.

## Lineage

**Bacara** — Encore's predecessor. The Node.js incarnation toured with Richie Hawtin and was used
for a Steve Reich tribute at IRCAM Paris.

**ARP / UNA** — collaborator with [Chris Brown / @midicake](https://github.com/midicake). Public
debut: Superbooth 2026.

**[midi.guide](https://midi.guide)** — contributor. CC/NRPN definitions for the gear I own are
mostly mine.
